# saya
Online RST doc platform

```sh
# 生成不带任何第三方包的虚拟环境, 名字为venv
$ virtualenv --no-site-packages venv

# 进入虚拟环境
$ source venv/bin/activate

# 安装依赖项
(venv)$ pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 退出虚拟环境
(venv)$ deactivate
```
